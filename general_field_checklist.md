## Comms

- [ ] Starlink unit.
- [ ] Starlink ethernet adapter.
- [ ] Re-activate Starlink subscription that should be on pause (ask Hadrien).
- [ ] WiFi router with WAN port and swappable antenna (SMA).
- [ ] TP-link router as wifi extender with a mast to go on the ute tray.
- [ ] 5+ m ethernet cable for the router.
- [ ] high-gain 2.4GHz WiFi antenna.



## Misc

- [ ] Android mobile devices (1 pp who does stage 4). These need good gyro and compass. Cheap devices (<$500) are usually not suitable for this task (Samsung: S series devices are okay, but not the A series). Most people prefer a tablet over a phone for this task.
- [ ] hand-held UHF radios (1 pp) + charger.


## Power
We generally run trips with two independant power sources for redundancy.

One generator, mostly for charging drone batteries, as that draws a lot of power:
- [ ] Generator. Recently serviced, spare parts.
- [ ] Petrol or diesel. Usage: ~15 litres / 12h.
- [ ] Power board.
- [ ] Long extension lead (15m+) to keep the noise at a distance.

And a battery + solar system (to have quiet time at night while still be able to run smaller things like laptops and Starlink):
- [ ] 2000Wh Battery station + cables (notably XT90 to Anderson plug connector).
- [ ] Roof-rack solar system: ~300W panel + MC4 fuse + MC4 extension leads + MC4 to Anderson adapter (all of this should be mounted permanently on Rhonda and not removed).
